2. IR2110 Mosfet Gate Driver IC (3n)
3. IRF740 Mosfet (6n)
4. MUR460 Diode (6n)
5. 400V 100uF Capacitor (1n)
6. 1N4007 Diode (21n)
7. 25V 470uF Capacitor (3n)
8. 15 0 Transformer 750mA (3n)
9. 0.22uF Capacitor (6n)
10. 10 ohm resistor (6n)
11. 10K ohm Potentiometer (1n)
